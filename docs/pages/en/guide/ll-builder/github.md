# App to store

## Main process

- Create a linyaps project and use ll-builder to build a linyaps app locally.

- Submit a pull request to the [https://github.com/linglongdev](https://github.com/linglongdev) repository.

- After the project maintainer approves the content, the build process is triggered. After the build is successful, the content will be submitted to the test repository of linyaps, and the GitHub action can obtain the build log and status.

- After the app test has passed, it will be put on the [linyaps Web Store](https://store.linyaps.org.cn/).
