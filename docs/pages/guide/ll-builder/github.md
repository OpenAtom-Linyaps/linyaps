# 上架应用到商店

## 主要流程

- 创建一个如意玲珑工程，使用ll-builder在本地构建出如意玲珑应用。

- 将如意玲珑工程以pull request（PR）方式提交到[https://github.com/linglongdev](https://github.com/linglongdev)项目。

- 项目维护者审核内容通过后，将触发构建流程，构建成功后内容将被提交到如意玲珑测试仓库。

- 应用通过测试后将会合并PR，并将应用上架到[如意玲珑网页商店](https://store.linyaps.org.cn/)。
