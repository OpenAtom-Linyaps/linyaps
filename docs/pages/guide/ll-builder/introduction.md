# ll-builder简介

`ll-builder`是为应用开发者提供的一款构建如意玲珑应用工具。

主要功能如下：

- 支持在独立容器内构建。

<!-- - 定义了一套版本管理系统。 -->

- 提供 DTK 开发套件。

<!-- - 包含完整推送发布流程。 -->

查看 `ll-builder`命令的帮助信息：

```bash
ll-builder --help
```

ll-builder 命令的帮助信息如下：

```text
如意玲珑构建工具 CLI
一个用于构建如意玲珑应用的命令行工具

用法: ll-builder [选项] [子命令]

Options:
  -h,--help                   打印帮助信息并退出
  --help-all                  展开所有帮助
  --version                   显示版本

Subcommands:
  create                      创建如意玲珑构建模板
  build                       构建如意玲珑项目
  run                         运行构建好的应用程序
  export                      导出如意玲珑layer或uab
  push                        推送如意玲珑应用到远程仓库
  import                      导入如意玲珑layer文件到构建仓库
  extract                     将如意玲珑layer文件解压到目录
  repo                        显示和管理仓库

如果在使用过程中遇到任何问题，
您可以通过此项目向如意玲珑项目团队报告错误：https://github.com/OpenAtom-Linyaps/linyaps/issues
```
