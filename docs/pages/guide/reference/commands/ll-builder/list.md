% ll-builder-list 1

## NAME

ll-builder-list - 列出已构建的应用程序

## SYNOPSIS

**ll-builder list** [*options*]

## DESCRIPTION

`ll-builder list` 命令用于列出已构建的应用程序。

## OPTIONS

**-h, --help**
: 打印帮助信息并退出

**--help-all**
: 展开所有帮助

## EXAMPLES

列出所有已构建的应用程序：

```bash
ll-builder list
```
